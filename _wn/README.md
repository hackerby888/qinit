# _wn

Qubic dynamic contract (`qinit new --template counter`).

```bash
qinit up        # sync headers + run a dev node
qinit dev       # watch contracts/_wn.h -> auto build+deploy on save
qinit call      # interactive: pick contract -> fn/proc
```

Config in `qinit.json` (name, contract, core, rpc). Slot is auto-allocated by name.
